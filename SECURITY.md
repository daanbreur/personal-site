# Security Policy

## Reporting a Vulnerability

Please do **NOT** report security vulnerabilities through public GitHub issues.

To report vulnerabilities contact me by email at: security@dnbr.cloud

You may encrypt sensitive reports using my PGP key:

https://dnbr.cloud/BF375AABF538ABCA6EE5040B6A0DDAB448DB05A8.asc

When possible, include:

* A description of the vulnerability
* Steps to reproduce it
* The affected page, component, or dependency
* The potential security impact
* Any suggested remediation

The security contactinformation for this project can also be found over at: https://dnbr.cloud/.well-known/security.txt
